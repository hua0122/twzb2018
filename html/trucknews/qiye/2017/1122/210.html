卡友的福音：全新升级美孚黑霸王超高级CK-4润滑油即将登陆中国

发布时间：2017-11-22 13:34:07 0

来源： 环球商用车

twzb

已关注
“2017年是全国上下全面执行国五排放标准的收获之年，借鉴了欧六排放标准的国六也将在2020年正式实施。全新升级的美孚黑霸王超高级15W-40的优越性能将帮助OEM和车队用户从容面对未来不断提升的需求，让大家的货运生意沿着不断演进的行业需求一路顺畅。”
 
 
近日，埃克森美孚携旗下即将全新升级的美孚黑霸王超高级15W-40亮相第七届中国国际柴油发动机峰会。作为美孚黑霸王在国内推出的首款API等级达到CK-4的产品，其在剪切稳定性、高低温保护、氧化控制、沉积物控制、空气释放性等方面都有大幅提升，受到了与会嘉宾的高度关注。该款产品的长效配方能有效帮助OEM和车队用户应对物流行业从“节能减排”到“降本增效”的需求提升，令在场设备制造商等行业专家期待不已。
 
 
埃克森美孚（中国）投资有限公司汽车制造商润滑油技术顾问刘振中先生向参会嘉宾介绍美孚黑霸王超高级15W-40说，“美孚黑霸王超高级15W-40的大量实验数据让我们引以为傲，这款产品在许多指标上优于API CK-4的规格要求，其抗磨损保护性能比 CK-4 API 发动机测试要求高 50%[ 所引用的数据和结果基于美国原始设备制造商设备、换油周期建议和现场测试。结果可能会因车辆／发动机型号和状况以及驾驶状况和环境条件的不同而存在差异。请在咨询原始设备制造商后再行延长换油周期，尤其是设备／车辆仍处于原始设备制造商保修期内时。]，能够简化卡车的维护保养，帮助广大OEM与车队用户有效节省日常维护成本。”

“纵观国内行业标准和全球行业标准的发展不难发现，全球汽车市场对整车发动机的要求已不仅限于‘减排’，对‘增效’也提出了更高的要求。”埃克森美孚（中国）投资有限公司副总经理杨东先生在接受采访时表示，“作为商用车润滑油领域的领先者，埃克森美孚一直紧跟全球行业标准变化，充分了解世界知名整车及发动机制造商的研发需求，凭借超过90年的研发经验、强大的技术团队，携手设备制造商实现创新突破。美孚黑霸王产品家族的这一全新成员的到来，将让我们更有自信为中国OEM、车队用户提供简化的车队润滑管理方案，帮助他们应对‘国五’以及今后更严格的排放标准，更帮助其有效节省日常维护成本，从‘节能减排’到‘降本增效’。”
